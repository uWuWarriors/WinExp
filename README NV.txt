Быстрая установка драйверов nvidia. Найдет оптимальный драйвер для системы(для всех видеокарт от зеленых). Есть возможность отключать shadowplay, physX и т.д. + есть возможность выставлять свои настройки для драйвера(приоритет, чистая установка, отключение телеметрии).
После установки стоит запустить furmark.
